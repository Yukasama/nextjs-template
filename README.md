# Nextjs Template

## Installation Steps

### 1. Install packages

```bash
pnpm i
```

### 2. Setup Prisma ORM

```bash
pnpm run prisma
```

### 3. Start server

```bash
pnpm run dev
```

## Features

- Next-Auth v5 (Authentication)
- SQLite (Dev DB)
- Prisma (ORM)
- Tailwind
- Resend (Mailing)
- Testing (Jest)
- Logging (Pino)
