# Next.js Template

## Preparation

```bash
# Install deps and playwright browsers
pnpm prepare

# Start the server
pnpm dev
```

## Features

- Next.js 15
- Tailwind v4
- Dark Theme (next-themes)
- End-To-End Tests (Playwright)
- Logging (Pino)
- Typesafe Environment Variables (t3-env)
- Formatting (Prettier)
- Linting (ESLint 9 + SonarQube)
- A+ Security Headers (CSP with Nonce)
- Docker Ready
- CI/CD with GitHub Actions for Linting, Testing, Security
