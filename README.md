# Nextjs Template

## Installation Steps

### 1. Install packages

```bash
pnpm i
```

### 2. Setup Prisma ORM

```bash
pnpm run prisma
```

### 3. Start server

```bash
pnpm run dev
```

## Features

- Next.js 14
- Next-Auth v5 (Authentication)
- SQLite (Dev DB)
- Prisma (ORM)
- Tailwind + shadcn-ui (Styling)
- Dark Theme (next-themes)
- Resend (Mailer)
- Jest (Tests)
- Pino (Logging)
- Zod (Validation)
- Prettier (Formatter)
- ESLint + SonarQube (Linter)
